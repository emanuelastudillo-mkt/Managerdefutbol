# Fútbol Manager MVP - V1.17

Paquete de actualización del proyecto para navegador.

## Cambios de esta versión
- Escudos limpios: se elimina el texto/abreviatura debajo del escudo.
- El anillo de estado físico queda centrado sobre el perímetro del círculo gris del jugador.
- El temporizador de 120 segundos se muestra como avance semanal: lunes a sábado, y el domingo se activa el partido.
- Se oculta información interna de jugabilidad:
  - tabla detallada de lesiones con porcentajes y duraciones;
  - escala de efectos del estado del campo.
