# Fútbol Manager MVP - V1.21

Paquete de actualización del proyecto para navegador.

## Cambios de esta versión
- La charla motivacional queda bloqueada durante 5 turnos después de usarse.
- El paso semanal ya no muestra segundero visible; ahora usa barra de progreso.
- Se siguen mostrando los días de la semana hasta llegar al domingo.
- El menú lateral muestra escudo del club junto al nombre, fecha y jornada.
- Las estadísticas de jugadores quedan separadas por división.
- Se oculta la leyenda de premios por división en la tabla.
- Se reduce el tamaño visual de la pizarra sin cambiar el tamaño de los círculos de jugadores.
- Se ajusta el aprovechamiento del espacio de la pizarra para reducir superposiciones.
