# Fútbol Manager MVP V2.26

Actualización incremental del manager de fútbol local.

## Cambios V2.26
- Para ofrecer un jugador propio a otros clubes, primero el club debe haberle pagado al menos un sueldo anual.
- El pago anual de sueldos marca internamente a los jugadores como habilitados para ser ofrecidos.
- Los jugadores libres contratados o compras que llegan al plantel deben esperar hasta cobrar al menos un sueldo antes de poder ser ofrecidos.
- En Táctica, la lista de titulares ahora muestra también la media general del jugador.
